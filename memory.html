Memory Game

This game is like the memory game from your childhood, but with a twist. It is compatible with screen readers. You can test your verbal and spatial memory by using the arrow keys to navigate and listen to what is on the other side of the cards, and try to make matches.

You can either use your mouse, or keyboard.

  • Select a color palette from the drop down menu.
  • Click the button Load Cards and the 60 seconds countdown will start.
  • Click the cards, 2 at a time, to reveal the color. Look for paris of 2, that both have the same color. There are 8 pairs.
  • Click the Start over button if you'd like to start over before the current game ends.
  • Use the tab key to navigate the game settings and the game board. It is a 4 by 4 board.
  • Choose a color palette from the drop down menu by pressing the up or down arrow keys.
  • Press the enter key to click the button Load Cards and the 60 seconds countdown will start.
  • Press the tab key to enter the game board.
  • Press the arrow keys or tab and shift-tab to navigate the board. When you come to a card, that card's color will be spoken.
  • Press the enter or space key to flip over the card. Listen for pairs of 2, that both have the same color. There are 8 pairs.
  • You will hear a notice when you have 30 seconds left.
  • Tab back to the Start over button if you'd like to start over before the current game ends. Otherwise, when you win or the countdown runs down, you will be left at the dropdown menu to choose a new color palette again.
